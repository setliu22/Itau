# Mix65 Generated Dataset

This directory contains the active generated train/test/validation splits from
the mix65 nearest-name strategy.

- `train.parquet`
- `test.parquet`
- `validation.parquet`
- `all_splits.parquet`
- `generation_metrics.json`
- `validation_generation_metrics.json`
- `selected_validation_config.json`

Derived audits and scratch candidate tables are intentionally not retained
here unless they are needed for an active run.
